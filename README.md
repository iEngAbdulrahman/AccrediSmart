# AccrediSmart
AccrediSmart is a web-based accreditation support system designed to help higher education institutions manage and streamline course accreditation workflows. The system supports faculty members, course instructors, quality assurance teams, and accreditation reviewers by organizing course evidence, managing learning outcome mappings, calculating course attainment, and generating structured FCAR reports.

# Overview
AccrediSmart was developed to reduce the manual workload involved in preparing accreditation files for frameworks such as ABET, NCAAA, and SAQF. Traditional accreditation work often depends on scattered documents, spreadsheets, manual calculations, and repeated reporting tasks. AccrediSmart provides a centralized digital workflow that makes the process more organized, traceable, and efficient.

The system helps users manage course data, upload required evidence, map course learning outcomes to program outcomes and student outcomes, define assessments, enter student grades, calculate attainment, and generate FCAR reports. It also includes AI-assisted recommendations to support course improvement, while keeping the final decision under instructor review.

# Features
- Course setup and management.
- Preloaded course information including CLOs, Bloom’s levels, NCAAA domains, PLO mappings, and SO mappings.
- Evidence management based on undergraduate course portfolio categories.
- CLO mapping to PLOs, SOs, Bloom’s levels, and NCAAA domains.
- Assessment creation and assessment-item management.
- CLO-to-assessment-item mapping.
- Student grade entry at the assessment-item level.
- Automatic CLO attainment calculation.
- FCAR report generation with attainment summaries, tables, and charts.
- AI-assisted recommendation drafting for course improvement.
- Instructor review and editing of AI-generated recommendations.
- PDF export for accreditation reporting.
- Dashboard and progress tracking for accreditation readiness.

# System Workflow
1- Create or select a course.
2- Upload and organize course evidence.
3- Review course learning outcomes.
4- Map CLOs to PLOs, SOs, Bloom’s levels, and NCAAA domains.
5- Define assessments and assessment items.
6- Link assessment items to the relevant CLOs.
7- Enter student grades.
8- Calculate CLO attainment automatically.
9- Generate the FCAR report.
10- Review recommendations and export the final report.

# Project Value
AccrediSmart improves the accreditation preparation process by reducing repetitive manual work, minimizing calculation errors, and improving the visibility of course performance data. It supports evidence-based course evaluation and helps faculty members prepare structured accreditation reports more efficiently.

The goal of the system is not to replace faculty judgment, but to support instructors and quality assurance teams with organized data, automated calculations, and clear reporting tools.
